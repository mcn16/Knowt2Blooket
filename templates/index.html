How to Use

  1. Open your flashcard set on Knowt
  2. Make sure it’s public
  3. Paste the URL above
  4. Adjust settings and click convert
  5. Inside Blooket, import the CSV file using the "CSV Import" feature
  6. Enjoy your new Blooket set!
